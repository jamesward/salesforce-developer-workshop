JavaScript in Visualforce

  • Load any JS Library
  • Use either a public CDN or upload assets to Salesforce
  • JavaScript remoting or JSON Apex Controllers

Apex Controller

global with sharing class OppRemoter {
    @RemoteAction
    global static List<Opportunity> findAll() {
        return [SELECT Id, Name FROM Opportunity];
    }
}

JavaScript Remoting

<apex:page sidebar="false" showheader="false" controller="OppRemoter">
    <script>
        Visualforce.remoting.Manager.invokeAction(
                '{!$RemoteAction.OppRemoter.findAll}',
                function(result, event) {
                    console.log(result);
                },
                {escape: true}
        );
    </script>
</apex:page>
